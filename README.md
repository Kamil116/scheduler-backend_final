## InnoScheduleBot

The InnoScheduleBot is a Telegram bot and a website made for notifications about the studying schedule. For students it provides a way to know if a lesson starts soon, and for administrators it provides an easy way to manage it.



To use the bot, simply click on the link below and message the bot ‘/start’ to begin. Then the bot will guide you through the process.
https://t.me/innopolis_schedule_bot

Or to use the website, click on the link below:
https://sowftware-lnm73ahmb-mohamed-nour1.vercel.app/?vercelToolbarCode=2fgu0OIMIfftWvh


## How to install

```bash
python -m venv venv
echo "aiogram==3.0.0b7" > requirements.txt
echo "python-dotenv==0.21.1" > requirements.txt
source venv/bin/activate
pip install -r requirements.txt
```

Frameworks used: Python, SQLite
